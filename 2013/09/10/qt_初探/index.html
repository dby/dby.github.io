QT 简介

Qt 是一个跨平台的 C++ 图形用户界面库，由挪威 TrollTech 公司出品，目前包括Qt， 基于 Framebuffer 的 Qt Embedded，快速开发工具 Qt Designer，国际化工具 Qt Linguist 等部分 Qt 支持所有 Unix 系统，当然也包括 Linux，还支持 WinNT/Win2k，Win95/98 平台。

这里有着更详细的资料

Ubuntu下安装

  1. ubuntu 下安装QT
    $ sudo apt-get install qt4-dev-tools qt4-doc qt4-qtconfig qt4-demos qt4-designer

    1. qt4-dev-tools 包含了Qt Assistant及Qt Linguist等工具，因此不需要单独安装这两个工具。
    2. qt4-doc 是帮助文档，包含了Qt中各个类库的详细说明以及丰富的例子程序。
    3. Qt Assistant 工具来打开阅读。
    4. qt4-qtconfig 是配置Qt环境的一个对话框，一般默认就行了，很少有必要去更改。
    5. qt4-demos 包含很多可以运行起来的可执行文件以及源代码。qt4-designer是用来设计GUI界面的设计器。

  2. 安装连接MySQL的驱动程序
    $ sudo apt-get install libqt4-sql-mysql

  3. 安装第三方QWT库
    $ sudo apt-get install libqwt5-qt4 libqwt5-qt4-dev
    注：安装完这些后，打开Qt Designer，就会发现左边的Widget列表里面多了”Qwt Widget”这一组；

  4. 安装集成开发环境Qcreator
    sudo apt-get install qcreator
