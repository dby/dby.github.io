MySQLdb在mac OS X中安装指南

  • 首先安装配置MySQL

  • 到MySQL官方网站下载对应版本的mysql,并安装

    • 配置路径

      打开终端,进入~/.bash_profile
      输入以下内容并保存，export PATH=$PATH:/usr/local/mysql/bin/

    • 测试 打开终端，输入mysql -u -p回车,直至出现welcome to the MySQL...
      如果想修改密码，用如下命令/usr/local/mysql/bin/mysqladmin -u root password
      测试

  • 安装配置MySQLdb

    • 下载MySQLdb库
    • 进入文件目录，运行python setup.py install

  • 安装中遇到的问题汇总

    • mysql_config not found
      解决方法，编辑源代码文件夹中的site.cfg文件，去掉#mysql_config = /usr/local/bin/mysql_config前面的注释#，修改后面的路径为你的mysql_config真正的目录就可以了。

    • Reason:image not found
      打开终端,输入以下命令即可解决。
      sudo ln -s /usr/local/mysql/lib/libmysqlclient.18.dylib /usr/lib/libmysqlclient.18.dylib
      sudo ln -s /usr/local/mysql/lib /usr/local/mysql/lib/mysql
